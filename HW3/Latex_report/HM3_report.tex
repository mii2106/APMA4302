\section{Problem 1}

\textbf{1)} 

For the equation:

\begin{equation}
-\nabla^2 u + \gamma u^p = f
\end{equation}

The exact solution is given by $u_{exact}(x,y)= A \exp\left(-\frac{(x-x_0)^2+(y-y_0)^2}{\sigma^2}\right)$. Accordingly, the source term is defined as $f = -\nabla^2 u_{exact} + \gamma u_{exact}^p$

The discretization is as follows:

\begin{eqnarray}
\frac{\partial^2 u}{\partial x^2} \approx \frac{u^{i-1,j} - 2u^{i,j} + u^{i+1,j}}{\Delta x^2} \\
\frac{\partial^2 u}{\partial y^2} \approx \frac{u^{i,j-1} - 2u^{i,j} + u^{i,j+1}}{\Delta y^2} 
\end{eqnarray}

Thus, the equation becomes:

\begin{equation}
  - \frac{u^{i-1,j} - 2u^{i,j} + u^{i+1,j}}{\Delta x^2} - \frac{u^{i,j-1} - 2u^{i,j} + u^{i,j+1}}{\Delta y^2} + \gamma (u^{i,j})^p = f
\end{equation}

Assuming a uniform grid with spacing $\Delta x = \Delta y = h$, the discretized equation is given by

\begin{equation}
      \frac{ 4u^{i,j} -u^{i-1,j} - u^{i+1,j} - u^{i,j-1}- u^{i,j+1}}{h^2} + \gamma (u^{i,j})^p = f  
\end{equation}

Accordingly, the residual at interior nodes is defined as:

\begin{equation}
      F^{i,j}(u) =    \frac{ 4u^{i,j} -u^{i-1,j} - u^{i+1,j} - u^{i,j-1}- u^{i,j+1}}{h^2} + \gamma (u^{i,j})^p - f = 0  
\end{equation}

And for boundary nodes, the residual is given by:

\begin{equation}
    F^{i,j}(u) = u^{i,j} - u_{exact}(x_i,y_i) = 0
\end{equation}

\textbf{2)} The Jacobian corresponds to the derivatives of the residual with respect to the nodal values of $u$. For interior nodes, the partial derivatives are given by:

\begin{eqnarray}
    \frac{\partial F^{i,j}}{\partial u^{i,j}} = \frac{4}{h^2} + \gamma p (u^{i,j})^{p-1}\\
    \frac{\partial F^{i,j}}{\partial u^{i+1,j}} = \frac{\partial F^{i,j}}{\partial u^{i-1,j}} = \frac{\partial F^{i,j}}{\partial u^{i,j+1}} = \frac{\partial F^{i,j}}{\partial u^{i,j-1}} = -\frac{1}{h^2}
\end{eqnarray}

And for boundary nodes:

\begin{equation}
    \frac{\partial F^{i,j}}{\partial u^{i,j}} = 1
\end{equation}
\begin{equation}
    J(u) = 1
\end{equation}

If all unknowns are assembled into a global vector $\mathbf{u}$, the Jacobian associated with the interior equations can be written as:

\begin{equation}
\mathbf{J}(\mathbf{u}) = \mathbf{A} + \gamma p \, \mathrm{diag}\!\left(\mathbf{u}^{[p-1]}\right),
\end{equation}
where $\mathbf{A}$ is the five point finite difference matrix associated with the operator $-\nabla^2$ (each row as $\begin{bmatrix} ...& -1 & ...&-1&4&-1&...&-1&...
    \end{bmatrix} $), and $\mathrm{diag}(\mathbf{u}^{[p-1]})$ is a diagonal matrix whose entries are $u_k^{p-1}$.

For boundary nodes, the corresponding rows of the Jacobian are replaced by rows of the identity matrix, since the boundary residual is simply $u_k-g_k=0$.
Thus, the global Jacobian is obtained by using the interior stencil rows for interior nodes and identity rows for boundary nodes.

\textbf{7)} Figure 3 shows the results for a linear case. The problem was run with the option \texttt{-rct\_linear\_f} for $\gamma=100$ and $p=3$ on a $65\times65$ grid. In this case, the right-hand side is defined as $f = -\nabla^2 u_{\text{exact}}$, omitting the reaction term $\gamma u_{\text{exact}}^p$.



\begin{figure}[h!]
	\centering
	\includegraphics[width=10cm]{Figures/f3.png}
	\vspace{-3.5mm}
	\caption{Results $\gamma = 100$ and $p=3$}
	\label{fig:rotProtocol}
\end{figure}

The obtained relative error was $\frac{\|u - u_{\text{exact}}\|_2}{\|u_{\text{exact}}\|_2} = 0.408$ which is significantly larger than in the full manufactured solution case ($5.24\times10^{-4}$). The numerical solution reaches a maximum amplitude of approximately $0.59$, compared to $1.0$ for $u_{\text{exact}}$.

This occurs because using \texttt{-rct\_linear\_f} corresponds to solving the problem$-\nabla^2 u + \gamma u^p = -\nabla^2 u_{\text{exact}}$ which is not consistent with the manufactured solution, since the nonlinear term $\gamma u^p$ on the left-hand side has no corresponding term in the right-hand side. As a result, the solver converges to a solution different from $u_{\text{exact}}$.

Compared to the linear case $\gamma=0$, the nonlinearity acts as an absorption term that reduces the amplitude of the solution. As $\gamma$ increases, this effect becomes stronger, which explains the difference in amplitude between $0.59$ and $1.0$.

Figure 4 and table 3 shows the performance and convergence behavior for $\gamma=100$ and $p=3$.

First, from the left panel (wall time vs grid size), the computational cost increases significantly with mesh refinement, as expected due to the quadratic growth in the number of unknowns. The runtime scales approximately as $\mathcal{O}(N)$ with respect to the number of degrees of freedom. Additionally, increasing the number of processors reduces the wall time, demonstrating parallel scalability. However, the speedup is sublinear, especially for finer grids, likely due to communication overhead and load imbalance.

From the middle panel (relative error vs grid size), the method exhibits a clear second-order convergence rate. The slope of approximately $-2.02$ confirms that the finite-difference discretization achieves the expected $O(h^2)$ accuracy. This validates the correctness of the implementation and the consistency of the numerical scheme.

Finally, the right panel (Newton iterations vs grid size) shows that the number of Newton iterations remains constant (equal to 8) across all grid sizes. This could indicates that the nonlinear convergence is mesh independent and that Newton's method performs robustly for this problem. It also suggests that the initial guess and solver configuration are appropriate.

Overall, the results demonstrate that the solver achieves optimal convergence rates, good parallel scalability, and stable nonlinear performance across different grid resolutions.
